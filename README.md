##  Project Overview
This repository contains a Sales and Customer Analysis dashboard created in Power BI. 

** Guided Learning Project:** This project was built as part of my continuous skill development in data analytics. It is a guided practice project based on an excellent tutorial by **Leila Gharani**. The dataset (dummy data) and the core structural concepts were provided by the tutorial's author. 
 *Original Tutorial:* [Power BI Tutorial For Beginners by Leila Gharani](https://youtu.be/c7LrqSxjJQQ)

##  Skills & Techniques Applied
While following this project, I gained hands-on experience and successfully applied the following Power BI functionalities:
- **Data Transformation (Power Query):** Cleaning raw data, splitting and merging columns, and assigning proper data types.
- **Data Modeling:** Establishing relationships between the Fact table (Invoice Data) and the Lookup table (Customer Master Data).
- **Time Intelligence & DAX:** Utilizing Quick Measures to calculate Year-over-Year (YoY) percentage changes and Year-to-Date (YTD) sales.
- **Data Visualization:** Building interactive clustered bar charts, map visuals for geographical sales distribution, line charts for trend analysis, and KPI cards.
- **Interactive Filtering:** Implementing slicers for dynamic date filtering across all report pages.

##  Tools Used
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
